# Copy Creator — 产品需求文档 (PRD)

## 1. 产品概述

| 项目 | 内容 |
|------|------|
| 产品名称 | Copy Creator |
| 产品定位 | PC 端效率辅助工具 |
| 产品形态 | 桌面小窗 + 侧边栏面板，关闭后驻留系统托盘 |
| 核心功能 | 剪切板管理、快捷短语、翻译 |
| 目标平台 | Windows 11、macOS |
| 产品模式 | 开源免费 |
| 设计规范 | iOS 风格（磨砂玻璃、圆角卡片、SF 字体） |

## 2. 目标用户

- 客服 / 运维（高频重复话术、快捷短语）
- 开发者（代码片段、命令速查、终端填充）
- 翻译 / 文案工作者（跨语言文本处理）
- 日常办公用户（剪切板历史回溯）

## 3. 功能规格

### 3.1 剪切板管理

| 项目 | 说明 |
|------|------|
| 记录类型 | 文本、图片 |
| 使用方式 | 点击列表条目一键粘贴到当前光标位置 |
| 清理策略 | 用户可选：保留一周 / 一个月 / 三个月，默认一个月 |
| 清理逻辑 | 超出时间窗口的记录自动删除 |
| 存储 | 本地 SQLite，无网络传输，无隐私风险 |
| 应用屏蔽 | 不做屏蔽（数据完全本地，不存在密码泄漏至云端的问题） |
| 搜索 | 支持对记录内容模糊搜索 |

### 3.2 快捷短语

| 项目 | 说明 |
|------|------|
| 组织方式 | 场景组（用户自建，可命名 / 编辑 / 删除） |
| 使用方式 | 打开悬浮窗 → 切换到快捷短语 Tab → 选择场景组 → 点击短语 → 粘贴到当前光标 |
| 编辑 | 在应用内增删改短语和场景组 |
| 存储 | 本地 SQLite |
| 搜索 | 支持对短语标题和内容模糊搜索 |

### 3.3 翻译

| 项目 | 说明 |
|------|------|
| 交互方式 | 用户进入翻译 Tab → 在输入框粘贴/输入文本 → 点击翻译 → 显示翻译结果 |
| 引擎选择 | 设置中可选默认引擎：AI 翻译 / 内置免费翻译 |
| AI 翻译 | 用户配置兼容 OpenAI 格式的 API 端点（地址 + Key + 模型名），不限厂商 |
| 内置免费翻译 | 接入国内可直连的免费翻译服务（百度翻译 / 有道翻译），无需梯子 |
| 离线行为 | 翻译需联网，断网时功能置灰并提示「翻译需要网络连接」 |
| 翻译缓存 | 已翻译过的文本本地缓存，离线时可复用 |
| 翻译历史 | 可查看翻译记录 |

**翻译处理流程：**

```
用户粘贴文本 → 点击翻译
  ├─ 默认引擎为 AI 翻译 且 已配置 API Key → 调用大模型翻译 → 显示结果
  ├─ 默认引擎为内置翻译 或 未配置 API Key → 调用免费翻译 API → 显示结果
  └─ 请求失败 → 提示错误信息
```

### 3.4 系统功能

| 功能 | 说明 |
|------|------|
| 窗口置顶 | 一键切换悬浮窗是否置顶 |
| 系统托盘 | 关闭窗口最小化到托盘，右键托盘图标可打开/退出 |
| 全局快捷键 | 用户可自定义唤起/隐藏悬浮窗的快捷键 |
| 主题切换 | 亮色 / 暗色模式，一键切换 |
| 开机启动 | 可设置是否开机自启 |

### 3.5 后续规划（Phase 2）

- 应用内直接输入文本并实时翻译
- 一键填充到终端（Windows Terminal、macOS Terminal）
- 快捷短语云同步（可选，需账号体系）
- 翻译引擎热切换（翻译结果页一键切换引擎重新翻译）

## 4. 数据模型

```
clipboard_records
  ├─ id              UUID        PK
  ├─ type            TEXT        'text' | 'image'
  ├─ content         TEXT/BLOB   文本内容或图片二进制
  ├─ source_app      TEXT        来源应用名称
  ├─ created_at      DATETIME    索引
  └─ 清理: 按 created_at 删除过期记录

phrase_groups
  ├─ id              UUID        PK
  ├─ name            TEXT        场景组名称
  ├─ sort_order      INTEGER     排序
  ├─ created_at      DATETIME
  └─ updated_at      DATETIME

phrases
  ├─ id              UUID        PK
  ├─ group_id        UUID        FK → phrase_groups
  ├─ title           TEXT        短语标题
  ├─ content         TEXT        短语内容
  ├─ sort_order      INTEGER     排序
  ├─ created_at      DATETIME
  └─ updated_at      DATETIME

translation_history
  ├─ id              UUID        PK
  ├─ source_text     TEXT        待翻译文本
  ├─ target_text     TEXT        翻译结果
  ├─ source_lang     TEXT        源语言（自动检测 / 手动指定）
  ├─ target_lang     TEXT        目标语言
  ├─ engine          TEXT        'ai' | 'builtin'
  ├─ created_at      DATETIME    索引
  └─ 缓存: 相同 source_text + target_lang + engine 命中缓存直接返回

settings
  ├─ key             TEXT        PK
  └─ value           TEXT
```

## 5. 界面结构

```
桌面常驻态（收起）:
┌────┐
│ 📋 │  ← 剪切板
│ 📝 │  ← 快捷短语
│ 🌐 │  ← 翻译
│    │
│ ⚙  │  ← 设置
│ ☀  │  ← 主题
│ 📌 │  ← 置顶
└────┘
  仅显示玻璃拟态图标栏 (~90px)，置顶悬浮于桌面

功能态（展开）:
┌────┬─────────────────────────────┐
│ 📋 │  [🔍 搜索]                  │  ← 面板标题 + 搜索
│    ├─────────────────────────────┤
│ 📝 │  内容列表 / 翻译输入区        │  ← 面板内容区 (~320px)
│    │  条目 1                     │
│ 🌐 │  条目 2                     │
│    │  条目 3                     │
│    │                             │
│    │                             │
│ ⚙  │                             │
│ ☀  │                             │
│ 📌 │                             │
└────┴─────────────────────────────┘
  点击左侧图标按钮 → 右侧滑出对应功能面板
  点击已激活按钮 → 收起面板，回到常驻态
  窗口整体宽度: ~420px（收起时 ~90px）
```

## 6. 非功能需求

| 类别 | 要求 |
|------|------|
| 启动速度 | 冷启动 < 2 秒 |
| 内存占用 | 常驻 < 80MB |
| 包体积 | < 20MB |
| 剪切板响应 | 复制后 < 100ms 完成记录 |
| 数据安全 | 所有数据仅存储于本地，无任何网络上传 |
| 跨平台一致性 | Windows 11 与 macOS 界面外观一致 |

## 7. 多语言

- 默认支持：简体中文、English
- 通过 i18n 框架预留扩展能力

## 8. 接口规划（后续细化）

| 接口 | 用途 | 说明 |
|------|------|------|
| AI 翻译 | 调用大模型翻译 | 兼容 OpenAI `/v1/chat/completions` 格式，用户配置地址+Key+模型名 |
| 内置免费翻译 | 无 Key 时的兜底翻译 | 百度翻译 / 有道翻译免费 API，国内直连 |
